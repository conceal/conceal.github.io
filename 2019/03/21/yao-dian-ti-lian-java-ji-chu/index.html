1. JAVA基本语法

编写Java程序时，应注意以下几点：

  • 大小写敏感：Java是大小写敏感的，这就意味着标识符Hello与hello是不同的。
  • 类名：对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么每个单词的首字母应该大写，例如 MyFirstJavaClass(驼峰命名法) 。
  • 方法名：所有的方法名都应该以小写字母开头。如果方法名含有若干单词，则后面的每个单词首字母大写（驼峰命名法）。
  • 源文件名：源文件名必须和类名相同。当保存文件的时候，你应该使用类名作为文件名保存（切记Java是大小写敏感的），文件名的后缀为.java。（如果文件名和类名不相同则会导致编译错误）。
  • 主方法入口：所有的Java程序由public static void main(String args[])方法开始执行。

2. JAVA编程格式

import java.util.Scanner;

public class Main {
    public static void main(String[] args) throws Exception{
        System.out.println(config.a + config.b);
        Scanner in = new Scanner(System.in);
        int a = in.nextInt();
        System.out.println(a);
    }
}

3. JAVA数据类型

3.1 基本数据类型

byte、char、short、int、long、float、double、boolean

数据类型 类别 位数 范围 默认值
byte 字节型 8位 -2^7 ～ 2^7-1 0
char 字符型 16位 \u0000 ～ \uffff (0~65535) 0
short 短整型 16位 -2^15 ～ 2^15-1 0
int 整型 32位 -2^31 ～ 2^31-1 0
long 长整型 64位 -2^63 ～ 2^63-1 0L
float 单精度浮点型 32位 -3.4 X 10^38 ～ 3.4 X 10^38 0.0f
double 双精度浮点型 64位 -1.7 X 10^308 ～ 1.7 X 10^308 0.0d
boolean 布尔类型 true、false false

3.2 数据类型转换规则

JAVA中的数据类型在定义时就已经确定了，不能随意改变。但是JAVA允许有限度的进行类型转换处理：自动类型转换、强制类型转换

3.2.2 强制类型转换

将高级数据转换成低级数据，只能用强制类型转换，通常在要求强制类型转换的变量名前用()括号上所要强制转换的类型符号。如：

short s；
byte b = (byte)s;

4. 直接量、常量与变量

  • 直接量：就是数据类型的数值，如int型：3就是直接量
  • 常量：特定标识符表示的直接量，赋值之后就不可以更改：
    [修饰符] final 数据类型 常量标识符 [= 直接量]
public final int a = 3;
  • 变量：存储数据的特定标识符，可以更改存储的数据：
    数据类型 常量标识符 [= 直接量]
int a = 3;

5. JAVA的输入输出方式

  • 输入：Java的输入是通过Scanner类实现的，在时用Scanner类时要在项目最顶端倒入Scanner类，如：import java.util.Scanner;
Scanner scan = new Scanner(System.in) // System.in是系统的System类的输入流
int a = scan.nextInt();

  • 输出：Java的最简单的输出方式：
    System.out.print("Hello World:" + a);
    注意“+”：+在里面充当连接符的角色，但是当两个基本数据类型的变量被+连接时，他们会进行加操作；字符串和数据类型变量被+连接时，数据类型变量会自动转换成字符串。
